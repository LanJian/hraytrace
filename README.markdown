HRayTrace
=========

A Simple Ray Tracer in Haskell

Features
--------

* Diffuse lighting
* Specular lighting
* Shadows (Simple obstructions)

Supported Shapes
----------------

* Spheres
* Cubes
* Planes

Pending Features
----------------
* Polygons
* Reflection
* Refraction
* Global illumination

Progress
--------
Super sampled at 256 rays / pixel
