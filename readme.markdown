HRayTrace
=========

A Simple Ray Tracer in Haskell

Features
--------

* Diffuse lighting
* Specular lighting
* Shadows (Simple obstructions)

Pending Features
----------------
* Polygons
* Reflection
* Refraction
* Global illumination

Progress
--------
Super sampled at 256 rays / pixel taking 190 secs
